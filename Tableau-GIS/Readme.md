# Tableau-GIS

## 什麼是地理資訊系統(Geographic Information System,GIS)?
是一門綜合學科，結合地理學與地圖學，已經廣泛的應用在不同的領域，是用於輸入、儲存、查詢、分析和顯示地理資料的電腦系統
目前 Tableau 提供部分GIS支援

而常被載入GIS的為Shapefile 格式(.shp)，下圖即為台灣的Shapefile格式
