# Tableau-spatialfunction

## 用Tableau進行空間函數應用
## 航班路線軌跡
* 使用空間函式:
MAKELINE & MAKEPOINT
MakeLine 可產生測地線，在距離遙遠時會顯示曲線，較近的時候(當地/城市規模資料)則會呈現直線狀態。
